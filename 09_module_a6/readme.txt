A6 :- Spotlight
The Core Functions/Features of this above project:-
1. The spotlight is Circluar, white, translucent, with blur edges
2. Mouse wheel up/pinch-out increases spotlight size; mouse wheel down / pinch-in decreses it.
3. Default spotlight size: 100px with minimum diameter as 10px and maximum diameter as 300px
4. Outside the image boundary, the spotlight must be hidden completely
