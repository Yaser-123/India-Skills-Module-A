A1 :- Dual-handle price slider
The Core Functions/Features of this above project:-
1. The range span of this above module:- from 0$ to 1000$ in step of 25.
2. the two drag handles will not cross or overlap at any pont (minimum of 25$ gap)
3. draggin either handle will update the displayed price value in real-time
4. default initial value: 300 left and 700 right
5. display live price difference label between the two values (eg.$400)
6. display a summary label showing % of total range covered (e.g 50% of range)
How To run:-
1. clone/donwload the code files
2. open index.html
3. install extention "live server"
4. click on "Go Live"
